# Smart-Gesture-Board

A real-time computer vision-based system that enables touchless drawing and writing using hand gesture recognition.

## 🚀 Features
- Hand gesture-based drawing
- Real-time webcam input
- Smooth virtual whiteboard experience
- Built for smart classrooms and presentations

## 🛠️ Tech Stack
- Python
- OpenCV
- MediaPipe
- NumPy

## ▶️ Run Project
```bash
pip install -r requirements.txt
python smart_gesture_board.py
